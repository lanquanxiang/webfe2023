# JS事件
# 1. 事件模型（JS事件机制）
## 1. 事件响应
1. （了解）事件三要素
2. （重点）如何实现事件响应
	1. 直接在标签中添加事件响应（优点：简单、明了；缺点：不够灵活，不易于维护）
	2. 获得元素，然后为元素的事件属性赋值，赋值一个函数（优点：灵活；缺点：事件只能触发一个）
	3. 获得元素，为元素添加事件监听（监听事件、触发函数）
3. 事件模型
	1. 捕获型：从根-->子
	2. 冒泡型（默认）
