# 2. 选择器
## 1. 常用选择器
1. 通配符选择器  * 清除浏览器默认样式
2. 元素选择器
3. id选择器  div#div2
4. class选择器（派生选择器 div.blue）
## 2. 组合选择器
1. 后代选择器 E F，选中F，F是E的后代
2. 父子选择器 E>F，选中F，F是E的儿子
3. 相邻兄弟选择器E+F，选中F，F是E的后续相邻兄弟
4. 普通兄弟选择器E~F，选中F，F是E的后续兄弟
5. 群组选择器E,F 选中E或F
6. 派生选择器E.F（class） 选中F，F必须满足E
## 3. 属性选择器
1. [attribute=value]
2. [attribute^=value]
3. [attribute$=value]
4. [attribute*=value]
## 4. 伪元素（虚拟元素）选择器
1. ::after
2. ::before   属性 content:""  content:url()
